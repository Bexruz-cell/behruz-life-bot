# Behruz Life Bot 🤖

Telegram-бот, ведущий канал от лица 14-летнего Бехруза из Самарканда, Узбекистан.
Генерирует живые посты через Mistral AI, подбирает фото, работает по расписанию 24/7.

---

## Структура проекта

```
behruz-life-bot/
├── bot/
│   ├── __init__.py
│   ├── main.py          # Точка входа, polling, автоперезапуск
│   ├── config.py        # Настройки из .env
│   ├── database.py      # SQLite: посты, настройки, события
│   ├── generator.py     # Генерация постов через Mistral AI
│   ├── handlers.py      # Все команды и inline-кнопки (24 функции)
│   ├── keyboards.py     # Inline-клавиатуры
│   ├── poster.py        # Публикация в канал
│   ├── scheduler.py     # APScheduler: авто-постинг, отчёты
│   ├── photo.py         # Поиск фото через Unsplash/Picsum
│   └── news.py          # RSS-новости
├── requirements.txt
├── render.yaml          # Конфиг для Render.com
├── .env.example
└── README.md
```

## Быстрый старт локально

### 1. Клонировать репозиторий
```bash
git clone https://github.com/Bexruz-cell/behruz-life-bot.git
cd behruz-life-bot
```

### 2. Установить зависимости
```bash
pip install -r requirements.txt
```

### 3. Создать .env файл
```bash
cp .env.example .env
```
Заполни `.env`:
```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
MISTRAL_API_KEY=ваш_ключ_mistral
CHANNEL_ID=@ai_life_tochka_com
OWNER_ID=ваш_telegram_id
```

### 4. Запустить
```bash
python -m bot.main
```

## Деплой на Render.com (24/7 бесплатно)

### Шаг 1 — Залить код на GitHub
(код уже залит через GitHub PAT автоматически)

### Шаг 2 — Создать сервис на Render
1. Зайди на [render.com](https://render.com) и зарегистрируйся
2. Нажми **New +** → **Background Worker**
3. Подключи GitHub аккаунт
4. Выбери репозиторий `behruz-life-bot`
5. Render автоматически увидит `render.yaml` и предложит конфиг

### Шаг 3 — Заполнить переменные окружения
В разделе **Environment** добавь:

| Ключ | Значение |
|------|----------|
| `TELEGRAM_BOT_TOKEN` | токен из BotFather |
| `MISTRAL_API_KEY` | ключ Mistral AI |
| `CHANNEL_ID` | @ai_life_tochka_com |
| `OWNER_ID` | твой числовой Telegram ID |

### Шаг 4 — Deploy
Нажми **Create Background Worker** — Render запустит бота на 24/7.

> ⚠️ На бесплатном тарифе Render Background Worker работает **без ограничений по времени** (в отличие от Web Service).

## Деплой на Replit (альтернатива)

1. Форкни репозиторий или загрузи файлы в Replit
2. В разделе **Secrets** добавь те же переменные окружения
3. В `.replit` укажи команду запуска: `python -m bot.main`
4. Используй **Always On** (платная функция) или **UptimeRobot** для поддержания работы
